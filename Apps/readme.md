# [HU] Külső motor és Kompatibilitási Útmutató
Ez a mappa a modernebb lemezkarbantartó binárisok befogadására szolgál.

### Telepítés:
Másold ide a `defrag.exe`, `defragres.dll` és `dfrgui.exe` fájlokat. 
A script automatikusan elhelyezi őket a `System32` és a `System32\hu-HU` mappákba is a maximális kompatibilitás érdekében.

### Kompatibilitási Mátrix:

| Forrás OS | Cél OS (Kompatibilis?) | Megjegyzés |
| :--- | :--- | :--- |
| Win 11 | Win 10 (Igen) | Win 7-en instabil lehet a kernel eltérés miatt. |
| Win 10 | Win 7 / 8 (AJÁNLOTT) | A legjobb stabilitás/hatékonyság arány. |
| Win 7 | Win XP (Nem) | Az XP még nem kezeli az újabb típusú szerviz hívásokat. |
| Win 98 | Win 95 (Igen) | A DOS alapú motorok felcserélhetőek. |

# [EN] External Engine & Compatibility Guide
This folder is for hosting modern disk maintenance binaries.

### Installation:
Copy `defrag.exe`, `defragres.dll`, and `dfrgui.exe` here. 
The script will automatically place them in both `System32` and `System32\en-US` for maximum compatibility.

### Compatibility Matrix:

| Source OS | Target OS (Compatible?) | Notes |
| :--- | :--- | :--- |
| Win 11 | Win 10 (Yes) | May be unstable on Win 7 due to kernel differences. |
| Win 10 | Win 7 / 8 (RECOMMENDED) | Best stability/efficiency ratio. |
| Win 7 | Win XP (No) | XP does not support newer service call types. |
